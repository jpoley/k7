title: "Health"
description: "Root and /health endpoints"
---

## Root

Endpoint: `GET /`

```bash
curl "$BASE/"
```

Example:

```json
{ "message": "K7 Sandbox API", "version": "x.y.z" }
```

## Health

Endpoint: `GET /health`

```bash
curl "$BASE/health"
```

Example:

```json
{ "status": "healthy" }
```
